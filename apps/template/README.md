# @cmp/template

This is the standard application template for the monorepo.

Use this as a starting point when creating new applications.

## Getting Started

1. Copy this directory to a new app: `cp -r apps/template apps/new-app`
2. Update the `name` field in `package.json` to `@cmp/new-app`
3. Run `pnpm install` at the root of the repo to update lockfiles and workspaces.
